Features:
User can search for an NBA player based on the player's name.
User can view every NBA player in a given season.
User can view every NBA player for a given team in a given season.
User can view every NBA player in a given season in ascending or descending order of a given player statistic.
User can view the current roster of any NBA team by clicking on their logo icon on the homepage.
The searchable information includes:
Name
Year
Team
Average minutes played per game
Average points per game
Average field goal percentage per game
Average 3-point field goal percentage per game
Average free-throw percentage per game
Average number of rebounds per game
Average number of assists per game
Average number of blocks per game
Average number of steals per game
